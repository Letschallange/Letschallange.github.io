# 프로젝트명 : Let's Challange(박진수)
## 메인컨셉 : 도전

- 게임유저는 게임을 플레이하면서 여러가지 장애물이 배치된 맵을 도전하게 되고 이를 통해 도전정신과 맵을 깼을때의 성취감을 느낄 수 있습니다.
  

### 서브 컨셉 1 : 다양성

- 맵에 다양한 장애물을 배치하여 보다 단조로운 맵 대신 다양성을 지닌 맵을 통해 유저들에게 재미를 선사하는것이 목표 입니다.

### 서브 컨세 2 : 성취감

- 장애물들을 배치해 맵을 구성할 시 밸런스 테스트를 통해 알맞는 난이도를 구성하여 플레이하는 유저가 맵을 클리어 했을 시 허무하지 않고 성취감을 느낄 수 있게 구성 할 예정 입니다.
